Forest Fire Prediction

Using Ridge Regression and Machine Learning

About the Dataset

This application utilizes the Algerian Forest Fire dataset, collected from June to September 2012. It encompasses weather observations and Fire Weather Indices (FWI) from two regions in Algeria: Bejaia and Sidi Bel-abbes.

The dataset plays a crucial role in training models to predict forest fires based on weather conditions. By analyzing factors like temperature, humidity, wind speed, and various FWI values, we can gain valuable insights and potentially mitigate fire risks.

Model and Scaler Details

This application employs a Ridge Regression model trained using cross-validation to enhance its generalization capabilities. The model utilizes a StandardScaler for data normalization, ensuring all features contribute equally during prediction.

Important Note: Columns with high correlation, namely 'DC' and 'BUI', were removed during the training process to prevent redundancy and improve model performance.
