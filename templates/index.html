Algerian Forest Fires Prediction

About the Dataset

The Algerian Forest Fires dataset includes data collected from two regions in Algeria, Bejaia and Sidi Bel-Abbes, during the period from June to September 2012. The dataset consists of several meteorological factors that influence forest fires.

We have used this dataset to build a predictive model for the Fire Weather Index (FWI), which is an indicator of the likelihood of a fire occurring.

Model and Methodology

This project uses a LassoCV (Least Absolute Shrinkage and Selection Operator with Cross-Validation) regression model. Lasso is a type of linear regression that uses shrinkage. Shrinkage is where data values are shrunk towards a central point, like the mean. Cross-validation is a technique for assessing how the results of a statistical analysis will generalize to an independent dataset.

We used cross-validation to tune the hyperparameters of the Lasso model, ensuring that it performs well on unseen data. Additionally, we removed highly correlated features ('DC' and 'BUI') to improve the model's performance and avoid multicollinearity.

Features Used

  • Temperature (°C)
  • Relative Humidity (%)
  • Wind Speed (km/h)
  • Rain (mm)
  • Fine Fuel Moisture Code (FFMC)
  • Duff Moisture Code (DMC)
  • Initial Spread Index (ISI)
  • Region
  • Classes
